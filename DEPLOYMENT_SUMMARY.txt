================================================================
  🚀 快速启动步骤
================================================================

第1步: 配置环境
  $ cp .env.example .env
  $ nano .env  # 填写腾讯云密钥

第2步: 安装依赖
  $ ./start.sh  # Linux/Mac
  或
  > start.bat   # Windows

第3步: 启动应用
  $ python app.py

第4步: 访问系统
  Web界面: http://localhost:5000
  健康检查: http://localhost:5000/api/health

================================================================
  🔑 必需的配置项
================================================================

腾讯云配置 (必需):
  TENCENT_SECRET_ID=你的密钥ID
  TENCENT_SECRET_KEY=你的密钥Key
  
获取地址: https://console.cloud.tencent.com/cam/capi

微信公众号配置 (可选):
  WECHAT_TOKEN=你的Token
  WECHAT_APP_ID=你的AppID
  WECHAT_APP_SECRET=你的AppSecret

================================================================
  🔌 API接口概览
================================================================

对话接口:
  POST /api/chat
  - 功能: 发送聊天消息
  - 自动调用相关MCP服务

MCP服务管理:
  GET  /api/mcp/services        # 列出所有服务
  POST /api/mcp/register         # 注册新服务
  POST /api/mcp/services/{name}  # 调用指定服务

系统接口:
  GET /api/health      # 健康检查
  GET /api/system/info # 系统信息

微信接入:
  GET  /wechat  # 验证服务器
  POST /wechat  # 处理消息

================================================================
  🐛 常见问题
================================================================

Q: 提示"腾讯元宝客户端未初始化"
A: 检查 .env 中的腾讯云密钥配置

Q: 端口被占用
A: 修改 .env 中的 PORT 配置

Q: 安装依赖失败
A: pip install --upgrade pip
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

Q: 微信验证失败
A: 检查Token配置，确保服务器可访问
